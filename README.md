## 🎯 Resumo das Atividades Recentes

- 🎓 Tecnóloga em Análise e Desenvolvimento de Sistemas, formada em 2026 pela Universidade Pitágoras Unopar Anhanguera
- 🏢 Fundadora da **FlowConnect** e co-fundadora da **TrioVision Tech**
- 🤖 Desenvolvo **automações via WhatsApp**, **agentes de IA** (Gemini, ChatGPT, Claude AI) e integrações via API com **N8N** e **Make**
- 💰 Criei o **FlowMoney**, sistema pessoal de controle financeiro com WhatsApp como interface principal: a IA interpreta texto, foto de comprovante ou PDF de extrato, registra a transação e envia alertas inteligentes, com dashboard web completo (gráficos, metas, orçamento)
- 📄 Desenvolvi uma **automação completa de emissão e assinatura de contratos** (Make + Autentique API), com fluxo sequencial de assinaturas, notificações automáticas e planilha de controle em tempo real
- 🎓 Como parte da TrioVision Tech, desenvolvi uma **plataforma de cursos online multi-tenant** para a Escola Gideão, com subdomínio, painel administrativo e identidade visual independentes por escola
- 🔧 Criei um **sistema completo de agendamento e disparo de mensagens** usando N8N como back-end e uma página web própria como front-end, sem exigir conhecimento técnico do usuário final
- 🤝 Desenvolvi um **chatbot de atendimento via WhatsApp** com menu interativo, fila de distribuição por setor e pesquisa de satisfação automática
